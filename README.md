## 設定変更画面
注：コマンドによる設定変更は残っていますが、今後サポートされることはなく、コマンド追加の予定もありません。<br>
ロビーでTabキーを押すと、部屋設定のテキストが設定画面に変化します。<br>
操作方法は以下の通りです。<br>
| キー | 動作 |
| :---: | ---- |
| Tab | 設定画面表示/非表示 |
| 十字キー上 | カーソルを上に移動 |
| 十字キー下 | カーソルを下に移動 |
| 十字キー右 | 選択中の項目を実行 |
| 十字キー左 | 一つ戻る |
